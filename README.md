# Skyscanner Backpack React App

## Overview

This project is a simple React web application built using Skyscanner's Backpack design system as part of the Forage Front-End Software Engineering Virtual Experience Program.

The application demonstrates the integration of Backpack UI components and basic React development practices.

## Features

* Flight Schedule page
* Interactive calendar component
* Continue button
* Responsive and clean user interface
* Built with React and Skyscanner Backpack

## Technologies Used

* React
* JavaScript (ES6+)
* Skyscanner Backpack Components
* CSS/SCSS

## Project Structure

```text
src/
├── App.js
├── App.scss
├── App.test.js
└── index.js
```

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

### Run Tests

```bash
npm test
```

## Learning Outcomes

* Setting up a React application
* Working with reusable UI components
* Integrating Backpack design system components
* Running and understanding automated tests
* Basic front-end application customization
